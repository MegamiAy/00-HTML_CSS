Protocolo - linguagem única, que todos conseguem falar

Arpanet foi dividida

NSFNET - estudantes
comercios -  ... comercios

Internetworking -> foi cortado e ficou Internet

	Markup - marcação
	Language - linguagem

HTML -> linguagem de marção hipertexto -> Conteúdo

					> imagens

CSS - 	Cascading - cascata		
	Style - estilo			
	Sheets - folhas			

CSS -> Folhas de estilo em cascata     -> Design

					> sombras
					> tamanhos
					> posicionamentos

JavaScript -> interatividade 	       -> Interações
